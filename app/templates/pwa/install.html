Fallen Birdy Form als App nutzen

Folgen Sie den untenstehenden Schritten, um die Webanwendung auf Ihrem Smartphone oder Tablet als App auf dem Homescreen abzulegen.

Android (Chrome)

  1. Öffnen Sie die Seite Fallen Birdy Form im Chrome-Browser.
  2. Tippen Sie auf das Drei-Punkte-Menü oben rechts und wählen Sie „App installieren“ oder „Zum Startbildschirm hinzufügen“.
  3. Bestätigen Sie den Dialog. Die App erscheint nun auf dem Homescreen und kann wie eine native Anwendung gestartet werden.

iOS (Safari)

  1. Öffnen Sie die Anwendung in Safari.
  2. Tippen Sie auf das Teilen-Symbol und wählen Sie anschließend „Zum Home-Bildschirm“.
  3. Optional können Sie den Namen anpassen. Tippen Sie auf Hinzufügen, um die App auf dem Home-Bildschirm zu platzieren.

Desktop

Auch am Desktop können moderne Browser eine Installation anbieten. Achten Sie auf eine Schaltfläche oder einen Installations-Hinweis in der Adressleiste.

Tipp: Stellen Sie sicher, dass beim ersten Aufruf eine stabile Internetverbindung besteht, damit alle Inhalte für die Offline-Verwendung geladen werden können.
